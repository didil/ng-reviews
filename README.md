# [ngReviews]

A product reviews site , built using:

## Client
Single Page Application :
- Angular.JS 1.2.16 
- Firebase.io for real time data sync
- Bootstrap 3 for the UI
- Karma / Mocha / Chai for testing

## Server 
API Backend :
- Rails 4.1
- Ruby 2.1
- Postgres DB
- Rspec for testing

Demo Live at http://ng-reviews.herokuapp.com
